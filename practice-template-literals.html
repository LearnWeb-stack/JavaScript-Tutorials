Practice Template Literals & String Interpolation

Instructions

What You'll Practice:

  • Using template literals (backticks `) to create strings
  • Inserting variables into strings using ${variable}
  • Creating HTML elements using template literals

How to Use:

  • Each practice section has a specific task and expected output
  • Fill in the input fields with your information
  • Click the "Generate" button to see both the code and the result
  • Compare your output with the example provided

Practice 1: Heading (h1) Tag

Task: Create a heading that says "Welcome, [your name]!" using template literals.

Expected Output Example: If you enter "John", it should generate:

Welcome, John!

Learning Goal: Practice using template literals to insert a variable into an h1 tag.


Practice 2: Paragraph (p) Tag

Task: Create a paragraph that says "I am [age] years old and I live in [location]." using template literals.

Expected Output Example: If you enter age "25" and location "New York", it should generate:

I am 25 years old and I live in New York.

Learning Goal: Practice using template literals to insert multiple variables into a p tag.




Practice 3: Combining Tags

Task: Create a personal introduction using both h1 and p tags. The structure should be:
  • An h1 tag saying "Hello, I'm [name]!"
  • A p tag saying "I really enjoy [hobby]."
  • Another p tag saying "Welcome to my page!"
Expected Output Example: If you enter name "John" and hobby "painting", it should generate:

Hello, I'm John!

I really enjoy painting.

Welcome to my page!

Learning Goal: Practice combining multiple HTML tags with template literals to create a structured content block.
