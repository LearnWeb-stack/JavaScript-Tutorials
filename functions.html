JavaScript Functions

From Basic to Intermediate Concepts

What are Functions?

Functions are reusable blocks of code that perform specific tasks. Think of them as mini-programs that you can call whenever you need them. Like a recipe in a cookbook, once you write a function, you can use it again and again without rewriting the same code.

Why Use Functions?
  • Code Reusability: Write once, use many times - saves time and reduces errors
  • Better Organization: Break down complex problems into smaller, manageable pieces
  • Maintainability: Easier to find and fix bugs when code is organized into functions
  • Modularity: Create independent, reusable pieces of code that work together
Key Terms to Know:
  • Function Declaration: The way you create a function
  • Parameters: Variables that functions receive as input
  • Return Value: The result that a function sends back
  • Function Call: When you use/execute a function

Basic Function Syntax Beginner

Understanding Function Structure:
  1. function keyword: Tells JavaScript you're creating a function
  2. function name: What you call your function (make it descriptive!)
  3. parameters: Input values the function will use (optional)
  4. function body: The code that runs when the function is called
  5. return statement: The value the function gives back (optional)

// Basic function structure
function functionName(parameter1, parameter2) {
    // Code to be executed
    return result;    // Send back a value
}

// How to call/use the function
const result = functionName(value1, value2);

Example 1: Adding Two Numbers

Purpose: This function takes two numbers and returns their sum.


// Function to add two numbers
function addNumbers(num1, num2) {
    return num1 + num2;    // Returns the sum of num1 and num2
}

// Using the function
const result = addNumbers(5, 3);  // result will be 8
How it works:
  1. Function receives two parameters: num1 and num2
  2. It adds these numbers using the + operator
  3. Returns the sum using the return statement
  4. When called with addNumbers(5, 3):
    • num1 gets the value 5
    • num2 gets the value 3
    • 5 + 3 is calculated
    • 8 is returned

Types of Functions Beginner

In JavaScript, there are several ways to create functions. Here are the two most common ways:

1. Function Declaration

Description: The traditional way to create a function. These are hoisted, meaning you can use them before they're defined in your code.


// Function Declaration
function greet(name) {
    return `Hello, ${name}!`;    // Uses template literal for string
}

// You can use it like this:
const greeting = greet("John");   // Returns: "Hello, John!"
Key Points:
  • Starts with the 'function' keyword
  • Must have a name
  • Can be used anywhere in your code
  • Good for main functions that you'll use throughout your code

2. Function Expression

Description: Assigns a function to a variable. These are not hoisted, so you must define them before using them.


// Function Expression
const greet = function(name) {
    return `Hello, ${name}!`;
};

// Using the function
const greeting = greet("Jane");   // Returns: "Hello, Jane!"
Key Points:
  • The function is assigned to a variable
  • The function itself doesn't need a name
  • Must be defined before you can use it
  • Good for functions that might be reassigned later

Basic Function Examples Beginner

Example: Calculate Area of Rectangle

Purpose: Calculate the area of a rectangle when given length and width.


// This function calculates the area of a rectangle
function calculateArea(length, width) {
    // Length and width are multiplied to get the area
    return length * width;
}

// Using the function
const area = calculateArea(5, 3);  // area = 15 (5 × 3)
How it works:
  1. Function takes two parameters: length and width
  2. Multiplies length by width to get area
  3. Returns the calculated area
  4. In the example:
    • length = 5
    • width = 3
    • area = 5 × 3 = 15

Example: Check Even Number

Purpose: Determine if a given number is even or odd.


// This function checks if a number is even
function isEven(number) {
    // Uses modulo operator (%) to check if number is divisible by 2
    return number % 2 === 0;  // Returns true for even numbers, false for odd
}

// Using the function
const checkNumber = isEven(4);  // Returns true because 4 is even
How it works:
  1. Takes a number as input
  2. Uses modulo (%) to get remainder when divided by 2
  3. If remainder is 0, number is even
  4. Returns true/false (boolean value)

Example: Nested Functions with Parameters

Purpose: Demonstrate how functions can be nested inside other functions and share parameters.


// Outer function that calculates total price with tax and discount
function calculateTotalPrice(price, discountPercent) {
    // Nested function to calculate discount amount
    function calculateDiscount(originalPrice) {
        return (originalPrice * discountPercent) / 100;
    }

    // Nested function to calculate tax (fixed 10%)
    function calculateTax(priceAfterDiscount) {
        const taxRate = 10;
        return (priceAfterDiscount * taxRate) / 100;
    }

    // Calculate final price
    const discount = calculateDiscount(price);
    const priceAfterDiscount = price - discount;
    const tax = calculateTax(priceAfterDiscount);
    
    return {
        originalPrice: price,
        discount: discount,
        tax: tax,
        finalPrice: priceAfterDiscount + tax
    };
}

// Using the function
const receipt = calculateTotalPrice(100, 20);  // 100 dollars with 20% discount
How it works:
  1. Outer function (calculateTotalPrice) takes price and discount percentage
  2. First nested function (calculateDiscount):
    • Uses discountPercent from outer function
    • Calculates discount amount
  3. Second nested function (calculateTax):
    • Calculates tax on discounted price
    • Uses fixed tax rate of 10%
  4. Returns object with all calculations

Intermediate Function Concepts Intermediate

Arrow Functions


// Regular function
function multiply(a, b) {
    return a * b;
}

// Arrow function
const multiply = (a, b) => a * b;

// Arrow function with single parameter
const square = x => x * x;

Function with Default Parameters


function greetUser(name = 'Guest', greeting = 'Hello') {
    return `${greeting}, ${name}!`;
}

const message1 = greetUser();  // "Hello, Guest!"
const message2 = greetUser('John', 'Hi');  // "Hi, John!"

Functions with Loops Intermediate

Calculate Sum of Array


function sumArray(numbers) {
    let sum = 0;
    for(let i = 0; i < numbers.length; i++) {
        sum += numbers[i];
    }
    return sum;
}

const numbers = [1, 2, 3, 4, 5];
const total = sumArray(numbers);  // total = 15

Find Maximum Number


function findMax(numbers) {
    let max = numbers[0];
    for(let number of numbers) {
        if(number > max) {
            max = number;
        }
    }
    return max;
}

const nums = [23, 55, 12, 89, 34];
const maxNum = findMax(nums);  // maxNum = 89
